## 🔍 关键信息速查

### 项目基本信息
- **项目名称**: 杀伤链体系效能评估系统
- **技术栈**: Vue 3 + Element Plus + Pinia + Canvas
- **当前进度**: 推演仿真基础功能完成（约70%）
- **紧急任务**: 网络连接可视化
- **GitHub**: https://github.com/lclclca/prodesign

### 文件位置
```
所有文件在: /mnt/user-data/outputs/
源码应放在: src/views/Simulation/index.vue
Store应放在: src/store/modules/network.js
```

### 核心概念
- **节点（Node）**: 传感器/打击单元/指挥中心
- **边（Edge）**: 探测/通信/打击关系
- **杀伤链**: 探测→决策→打击→评估
- **效能评估**: 量化推演结果

### 最紧急的事
1. 实现网络连接线的绘制（Task 1）
2. 实现杀伤链分析（Task 2）
