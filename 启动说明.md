# 前端项目启动说明

## 快速开始

### 1. 安装Node.js
确保已安装 Node.js 18+ 和 npm 9+

检查版本：
```bash
node -v
npm -v
```

### 2. 解压项目
```bash
tar -xzf frontend.tar.gz
cd frontend
```

### 3. 安装依赖
```bash
npm install
```

如果下载慢，可以使用国内镜像：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### 4. 启动开发服务器
```bash
npm run dev
```

启动后访问：http://localhost:3000

### 5. 默认登录账号
- 用户名：admin
- 密码：admin123

## 开发指南

### 添加新页面

1. 在 `src/views/` 下创建页面组件
2. 在 `src/router/index.js` 中添加路由
3. 如需要，在侧边栏添加菜单项

### 调用API

```javascript
import { getEquipmentList } from '@/api/equipment'

const loadData = async () => {
  try {
    const data = await getEquipmentList({ page: 1 })
    console.log(data)
  } catch (error) {
    ElMessage.error('加载失败')
  }
}
```

### 使用Store

```javascript
import { useUserStore } from '@/store/modules/user'

const userStore = useUserStore()
console.log(userStore.username)
```

### 格式化数据

```javascript
import { formatDateTime } from '@/utils/format'

const time = formatDateTime(new Date())
```

## 常见问题

### Q: 启动后页面空白？
A: 检查浏览器控制台是否有错误，确保依赖安装完整

### Q: API请求失败？
A: 确保后端服务已启动，检查代理配置

### Q: 登录后跳转失败？
A: 检查Token是否正确保存，路由配置是否正确

### Q: 图表不显示？
A: 确保容器有高度，在mounted后初始化图表
